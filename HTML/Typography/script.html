What exactly are Script fonts?

Script fonts, otherwise known as Handwriting fonts, are fonts that look like peoples handwriting or in some cases like fonts that are hand-written in those fancy scripts. (I'm sure you could have guessed that by the name ðŸ˜‰).

Similar to Monospace fonts, Script fonts shouldn't be overused as it may make your page look very messy or "amateur-like".

The way that the above example uses Script font is well done. The design of the website looks as though it's been drawn. To have a Script font compliment the design is perfect.

Generally, as a rule of thumb, you should stick to Serif and Sans-Serif fonts when creating websites. I'm not here to boss you around but make sure you think the website looks good when you use that script font!
